**Step 2: Verify the MCP tools work** by calling the mcp-proxy directly with curl (since restarting Claude Code to pick up the config change is a user action):

```bash
# Test the 16 named MCP tools exposed by alphahub-mcp
MCP_SECRET="${MCP_PROXY_SECRET}"
NEW_URL="https://qcunascacayiiuufjtaq.supabase.co"

for tool in list_clients get_client_detail search_clients get_billing_summary get_ad_spend_overview get_campaign_health get_lead_pipeline get_communications get_financial_projections get_alerts get_stripe_balance get_stripe_customers get_stripe_invoices get_stripe_subscriptions get_stripe_charges get_stripe_payouts; do
  status=$(curl -s -o /dev/null -w "%{http_code}" -X POST \
    "${NEW_URL}/functions/v1/mcp-proxy" \
    -H "Content-Type: application/json" \
    -H "x-mcp-secret: ${MCP_SECRET}" \
    -d "{\"tool\": \"${tool}\", \"params\": {}}")
  echo "${tool}: HTTP ${status}"
done
```

All 16 tools should return HTTP 200. If any return errors, investigate and fix.

**Step 3: Count total available tools** by calling the list_tools meta-tool:
```bash
curl -s -X POST "${NEW_URL}/functions/v1/mcp-proxy" \

  -H "x-mcp-secret: ${MCP_SECRET}" \
  -d '{"tool": "list_tools", "params": {}}' | jq '.result | length'
```
