  <name>Task 2: Convert verify-onboarding-live to Anthropic tool calling format</name>
  <files>alphahub-v2/supabase/functions/verify-onboarding-live/index.ts</files>
  <action>
  Read the current code for `verify-onboarding-live/index.ts` first. This function uses OpenAI-style tool calling with an `extract_form` tool. Convert to Anthropic tool calling format.

  **Conversion changes:**

  1. **Env var**: `Deno.env.get("LOVABLE_API_KEY")` -> `Deno.env.get("LLM_API_KEY")`
  2. **URL**: `https://ai.gateway.lovable.dev/v1/chat/completions` -> `https://api.anthropic.com/v1/messages`
  3. **Headers**: Same as Pattern A (x-api-key, anthropic-version, Content-Type)
  4. **Request body -- model + max_tokens**: `model: "claude-sonnet-4-6"`, add `max_tokens: 4096`
  5. **Request body -- system prompt**: Move system message from messages array to top-level `system` parameter
  6. **Request body -- tools array**: Change from OpenAI format to Anthropic format:
     - OpenAI: `{ type: "function", function: { name: "extract_form", description: "...", parameters: {...} } }`
     - Anthropic: `{ name: "extract_form", description: "...", input_schema: {...} }`
     - The schema content (properties, required, etc.) stays the same -- just the wrapper changes
  7. **Request body -- tool_choice**: Change from OpenAI format to Anthropic format:
     - OpenAI: `{ type: "function", function: { name: "extract_form" } }`
     - Anthropic: `{ type: "tool", name: "extract_form" }`
  8. **Response parsing**: Change from OpenAI format to Anthropic format:
     - OpenAI: `result.choices[0].message.tool_calls[0].function.arguments` (JSON string, needs JSON.parse)
     - Anthropic: `result.content.find(b => b.type === "tool_use").input` (already parsed object, NO JSON.parse needed)
     - Remove any `JSON.parse()` call on the tool arguments -- Anthropic returns parsed JSON directly

  **What NOT to change:**
  - All surrounding business logic (HTML fetching, form submission, result processing)
  - CORS handling
  - Database operations
  - Error messages and logging patterns
