<objective>
Add push notifications with deep linking and badge management. Register for APNs, store device tokens, handle foreground/background notification display, route notification taps to the correct screen, show unread badge on the chat tab, and manage the app icon badge.

Purpose: Push notifications are the sticky mechanism -- they bring users back to the app when they have new messages, payment reminders, or other events. Deep linking ensures tapping a notification lands on the right screen. The unread badge on the chat tab provides a persistent visual cue.

Output: 3 new Swift files (NotificationModels, PushNotificationManager, AppDelegate+Push) + 4 modified files (AppRouter, FloatingTabBar, ClientTabView, AlphaHubApp) + project.pbxproj update

    **NotificationModels.swift** -- Define notification-related types:

    1. `PushNotificationType`: String enum (chat, billing, course, lead, wallet_low). Each case has a `threadIdentifier: String` (for grouping: "chat", "billing", "courses", "leads", "billing") and a `categoryIdentifier: String` (for action buttons: "CHAT_MESSAGE", "BILLING_REMINDER", etc.).

    2. `DeviceTokenRecord`: Codable, Sendable struct for the device_tokens table. Fields: id (String?), userId (String), deviceToken (String), platform (String = "ios"), createdAt (String?), updatedAt (String?). CodingKeys for snake_case.

    **PushNotificationManager.swift** -- `@Observable @MainActor final class PushNotificationManager`:

    - Singleton: `static let shared = PushNotificationManager()` (needed for AppDelegate callback access)
    - Properties:
      - `isOnChatScreen: Bool = false` -- set true when ChatView appears, false on disappear. Used to suppress in-app chat notifications.
      - `notificationPermissionGranted: Bool = false`
      - private `router: AppRouter?` -- set via configure(router:) method
      - private `supabase: SupabaseClient`
    - `configure(router: AppRouter)`: Store router reference for deep linking. Call from AlphaHubApp.
    - `requestPermission() async`:
      - `UNUserNotificationCenter.current().requestAuthorization(options: [.alert, .badge, .sound])`
      - If granted, call `await UIApplication.shared.registerForRemoteNotifications()` on main actor
      - Update `notificationPermissionGranted`
    - `registerNotificationCategories()`:
      - Create `UNNotificationCategory` for "CHAT_MESSAGE" with `UNTextInputNotificationAction` (identifier: "REPLY", title: "Reply")
      - Create categories for "BILLING_REMINDER", "COURSE_UPDATE", "LEAD_NEW" (no custom actions, just for grouping)
      - `UNUserNotificationCenter.current().setNotificationCategories(categories)`
    - `saveDeviceToken(_ tokenString: String) async`:
      - Get userId from `supabase.auth.session.user.id`
      - Upsert to device_tokens table: `supabase.from("device_tokens").upsert([...], onConflict: "user_id, device_token")`
      - Silently fail if table doesn't exist yet (try? -- server-side setup may not be complete)
    - `handleNotificationTap(userInfo: [AnyHashable: Any])`:
      - Parse "type" from userInfo
      - Switch on type: "chat" -> router.clientTab = .chat; "billing" -> router.clientTab = .wallet; "course" -> router.clientTab = .courses; "lead" -> router.clientTab = .home
      - Clear app badge: `UIApplication.shared.applicationIconBadgeNumber = 0` (deprecated but still works; or use `UNUserNotificationCenter.current().setBadgeCount(0)` for iOS 16+)
    - `clearBadge() async`:
      - `try? await UNUserNotificationCenter.current().setBadgeCount(0)`

    **AppDelegate+Push.swift** -- UIApplicationDelegate:

    ```swift
    import UIKit
    import UserNotifications

    class AppDelegate: NSObject, UIApplicationDelegate, UNUserNotificationCenterDelegate {

        func application(_ application: UIApplication, didFinishLaunchingWithOptions ...) -> Bool {
            UNUserNotificationCenter.current().delegate = self
            return true
        }

        func application(_ application: UIApplication, didRegisterForRemoteNotificationsWithDeviceToken deviceToken: Data) {
            let tokenString = deviceToken.map { String(format: "%02x", $0) }.joined()
            Task {
                await PushNotificationManager.shared.saveDeviceToken(tokenString)
            }
        }

        func application(_ application: UIApplication, didFailToRegisterForRemoteNotificationsWithError error: Error) {
            print("[Push] Failed to register: \(error.localizedDescription)")
        }

        // Foreground notification handling
        func userNotificationCenter(_ center: UNUserNotificationCenter, willPresent notification: ...) {
            let userInfo = notification.request.content.userInfo
            let type = userInfo["type"] as? String

            if type == "chat" && PushNotificationManager.shared.isOnChatScreen {
                // Suppress -- messages appear via WebSocket
                completionHandler([])
            } else {
                completionHandler([.banner, .sound, .badge])
            }
        }

        // Notification tap handling (deep link)
        func userNotificationCenter(_ center: UNUserNotificationCenter, didReceive response: ...) {
            let userInfo = response.notification.request.content.userInfo
            Task { @MainActor in
                PushNotificationManager.shared.handleNotificationTap(userInfo: userInfo)
            }
            completionHandler()
        }
    }
    ```
