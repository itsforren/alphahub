      - Left: "Upcoming" label (captionSmall, AppColors.warning)
      - Center: amount in abbreviated currency (bodyLarge, textPrimary)
      - Right: due date in relative format (captionSmall, textTertiary) e.g. "in 3 days"
    - Style: subtle horizontal divider above, no heavy card -- just a clean info row that fits the hero card's aesthetic
    - If no upcoming payment exists, hide this row entirely (no empty state)

    - Handle zero wallet gracefully: show "$0" balance, 0% progress, no crash

    Accept wallet data, totalDeposits, displayedSpend, remainingBalance as parameters (or read from DataManager directly). Use `@Environment(DataManager.self)` if cleaner.
