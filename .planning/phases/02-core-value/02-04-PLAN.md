    **CampaignSpendChart.swift** -- Campaign spend visualization using Apple Swift Charts (import Charts). Uses `@Environment(DataManager.self)`.

    Layout:
    - SectionHeader("Campaign Spend") at top

    - Date range info: "Last 30 days" label (caption, textTertiary). NOTE: Date range filtering is a future enhancement -- for v1, default to last 30 days of available data.
    - The chart itself:

    ```swift
    Chart(chartData) { point in
      AreaMark(
        x: .value("Date", point.dateValue),
        y: .value("Spend", point.cost)
      )
      .foregroundStyle(
        LinearGradient(
          colors: [AppColors.accent.opacity(0.3), AppColors.accent.opacity(0.01)],
          startPoint: .top,
          endPoint: .bottom
        )
      )
      .interpolationMethod(.catmullRom)

      LineMark(
        x: .value("Date", point.dateValue),
        y: .value("Spend", point.cost)
      )
      .foregroundStyle(AppColors.accent)
      .lineStyle(StrokeStyle(lineWidth: 2))
      .interpolationMethod(.catmullRom)
    }
    .chartXAxis {
      AxisMarks(values: .stride(by: .day, count: 7)) { value in
        AxisValueLabel(format: .dateTime.month(.abbreviated).day())
          .foregroundStyle(AppColors.textTertiary)
      }
    }
    .chartYAxis {
      AxisMarks { value in
        AxisValueLabel()
          .foregroundStyle(AppColors.textTertiary)
      }
    }
    .frame(height: 200)
    ```

    - Below chart: summary stats row showing "Avg/day: $X" and "Target/day: $X" in caption, textSecondary. Compute avg from chartData sum / count. Target from clientProfile.targetDailySpend.
    - Computed `chartData`: filter adSpendDaily by selectedCampaignId (if not nil), then aggregate by date (sum costs for same date across campaigns if "All" is selected). Sort by date ascending. Limit to last 30 entries.
    - Handle empty data: if chartData is empty, show "No spend data available" message instead of empty chart. Swift Charts can crash or show nothing with zero data points.
    - Handle single data point: show a PointMark instead of LineMark for a single day.
