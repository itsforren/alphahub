    **DataManager.swift** -- Create `@MainActor @Observable final class DataManager`. Properties: clientProfile (ClientProfile?), wallet (ClientWallet?), totalDeposits (Double), trackedSpend (Double), performancePercentage (Double), adSpendDaily ([AdSpendDaily]), leads ([Lead]), campaigns ([Campaign]), billingRecords ([BillingRecord]), paymentMethods ([PaymentMethod]), isInitialLoad (Bool = true), isRefreshing (Bool = false), error (String?).

    Computed properties: `displayedSpend` = trackedSpend * (1 + performancePercentage / 100). `remainingBalance` = totalDeposits - displayedSpend. `firstName` = clientProfile?.name split by space, first component. `walletMetrics` returning WalletDisplayMetrics. `dashboardMetrics` returning DashboardMetrics computed from leads + clientProfile.

    Methods:
    - `loadAllData()` async -- Get userId from `supabase.auth.session.user.id`. Query clients table `.eq("user_id", value: userId.uuidString).single()` to get profile. Then `withTaskGroup(of: Void.self)` to parallel-fetch: fetchWalletData(clientId:), fetchBillingRecords(clientId:), fetchLeads(agentId:), fetchAdSpendDaily(clientId:), fetchCampaigns(clientId:), fetchPaymentMethods(clientId:), fetchPerformancePercentage(). Set isInitialLoad = false after.
    - `refreshAll()` async -- Set isRefreshing = true, call loadAllData(), set isRefreshing = false.
    - `fetchWalletData(clientId:)` async -- Get wallet from client_wallets, sum wallet_transactions deposits, sum ad_spend_daily cost since tracking_start_date. CRITICAL: This is the wallet balance computation. Follow the exact pattern from 02-RESEARCH.md Pattern 3.
    - `fetchBillingRecords(clientId:)` async -- From billing_records, filter non-archived, order by created_at desc.
    - `fetchLeads(agentId:)` async -- From leads, eq("agent_id"), order by lead_date desc. NOTE: uses agent_id NOT client_id.
    - `fetchAdSpendDaily(clientId:)` async -- From ad_spend_daily, eq("client_id"), order by spend_date desc, limit to last 90 days.
    - `fetchCampaigns(clientId:)` async -- From campaigns, eq("client_id").
    - `fetchPaymentMethods(clientId:)` async -- From client_payment_methods, eq("client_id").
    - `fetchPerformancePercentage()` async -- From onboarding_settings, eq("setting_key", "performance_percentage"), decode setting_value as String, convert to Double. Default to 0 if RLS blocks.

    Use `SupabaseConfig.client` (already exists). Each fetch method should use `do/catch` internally with `try?` so one failure doesn't block others. Store the supabase client as a `private let` property, defaulting to `SupabaseConfig.client`.
