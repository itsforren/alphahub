    **BillingView.swift** -- Main billing screen. Uses `@Environment(DataManager.self)`.

    Structure:
    ```
    ScrollView {
      LazyVStack(spacing: AppSpacing.lg) {
        // Filter bar at top
        BillingFilterBar(selectedFilter: $selectedFilter)

        // Payment methods section
        PaymentMethodCards(...) // will be added in Task 2

        // Transaction list grouped by month
        ForEach(groupedRecords.keys.sorted().reversed(), id: \.self) { monthKey in
          Section {
            ForEach(groupedRecords[monthKey]!) { record in
              BillingTransactionRow(record: record)
                .onTapGesture {
                  selectedRecord = record
                  showDetail = true
                }
            }
          } header: {
            // Month header: "March 2026", "February 2026"
            Text(monthKey)
              .font(AppTypography.heading3)
              .foregroundColor(AppColors.textPrimary)
              .frame(maxWidth: .infinity, alignment: .leading)
          }
        }

        // Empty state
        if filteredRecords.isEmpty && !dataManager.isInitialLoad {
          // Empty billing state: friendly message
          VStack(spacing: AppSpacing.md) {
            Image(systemName: "doc.text.magnifyingglass")
              .font(.system(size: 48))
              .foregroundColor(AppColors.textTertiary)
            Text("No transactions found")
              .font(AppTypography.bodyLarge)
              .foregroundColor(AppColors.textSecondary)
          }
          .padding(.top, AppSpacing.xxl)
        }
      }
      .padding(.horizontal, AppSpacing.screenPadding)
    }
    .refreshable {
      await dataManager.refreshAll()
      HapticManager.notification(.success)
    }
    .sheet(isPresented: $showDetail) {
      if let record = selectedRecord {
        TransactionDetailSheet(record: record)
          .presentationDetents([.medium])
          .presentationDragIndicator(.visible)
      }
    }
    .background(AppColors.pureBlack)
    ```

    State: `@State private var selectedFilter: BillingFilter = .all`, `@State private var selectedRecord: BillingRecord?`, `@State private var showDetail = false`.

    Computed `filteredRecords`: filter dataManager.billingRecords by selectedFilter (.all returns everything, .adSpend filters billingType == "ad_spend", .management filters billingType == "management").

    Computed `groupedRecords`: group filteredRecords by month string from createdAt (e.g., "March 2026"). Use DateFormatter with "MMMM yyyy" format. Return `[String: [BillingRecord]]` dictionary.

    When `dataManager.isInitialLoad`, show shimmer placeholders (6-8 shimmer rows mimicking transaction rows).

    **BillingFilterBar.swift** -- Three-segment filter using SwiftUI Picker with `.pickerStyle(.segmented)`. Define `BillingFilter` enum: `.all`, `.adSpend`, `.management` with display labels "All", "Ad Spend", "Management". Binding to selectedFilter.

    Style the segmented control to fit the dark theme. Use `.tint(AppColors.textPrimary)` or custom appearance via UISegmentedControl.appearance() in an onAppear block if needed for dark theme contrast.

    **BillingTransactionRow.swift** -- Single transaction row. Shows:
    - Left side: amount (heading3 font, textPrimary, abbreviated currency) and billing type label below ("Ad Spend" or "Management" in caption, textSecondary)
    - Right side: StatusPill for status (green .paid, yellow .pending, red .overdue or .failed) and relative date below (caption, textTertiary)
    - Full width, with horizontal padding
    - Subtle bottom divider (AppColors.divider, 0.5pt)
    - No heavy card borders -- rows flow in a clean list

    Accept `BillingRecord` as parameter.

    **TransactionDetailSheet.swift** -- Half-sheet (.medium detent) with transaction details. Dark background (surfaceElevated). Shows:
    - Amount (heroMedium size, centered)
    - Status pill
    - Detail rows in a VStack:
      - Type: "Ad Spend" or "Management"
      - Date: full formatted date (not relative)
      - Billing period: start - end (if available)
      - Due date (if available)
      - Paid date (if available)
      - Stripe account label
    - Each detail row: HStack with label (caption, textSecondary) and value (body, textPrimary)
    - "View on Web" button at bottom that opens alphaagent.io in Safari (generic link since we don't have per-transaction URLs)
